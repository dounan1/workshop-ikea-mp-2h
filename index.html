0. Welcome aboard!

1. Let's set up

  1. Open and bookmark the WeChat Official Accounts Platform
  2. Download the Integrated Development Environment (IDE) for WeChat: click here and choose it for Windows or Mac (Linux not available)
  3. Install the IDE on your computer
  4. Open the IDE to setup your first project
    • On the home screen, scan the QR code with your personal WeChat. You should read "Scan succesfully". On your mobile, authorize the login. You're in!
    • Choose the left menu (purple block) "Mini Program Project"
    • Click the + button on the bottom right of the window to add a new project
      1. Select a directory where the Mini Program code will be stored on your computer. (Suggestion: make a new folder on your Desktop)
      2. In the AppID field, you're supposed to input your registered AppID. Don't worry, you can use a "test account" by clicking on the blue link "Mini Program" just below the field.
      3. Select a name for this project (for your usage only, it won't show to users).
      4. If your project directory was empty, the WeChat IDE will provide you a "quick start template" with basic code. The option is selected by default. It is suggested to begin with this option activated.

2. Add some content and WXML markup

  • To save time, just right-click the following image and save it inside your WeChat Mini Program code directory (put this file inside a new "img" folder)
  • Then start writing some WXML code in index.wxml. We want:
    1. One image tag
    2. Two text tags
    3. One button tag
  • Have a little doubt here? no worries, always refer to the Official Documentation!
  • Your code will look like this:

  <image src="/img/kungfupanda.jpg"/>
  <text>Get new superpowers</text>
  <text>Learn to code!</text>
  <button type="default">Start today</button>

Tips

  • Use the built-in auto complete function to make sure you don't forget any end tags
  • Your image src allows to load the file from the current page (index.wxml), you want to start with a slash / to write an absolute path
  • Don't forget to place your images inside a folder, e.g /img
  • the text tag is "inline", i.e each tag will be side by side. If you want to break on two or multiple lines, consider using view tags instead

3. CSS for fonts & colors

  • First, transform your image tag into a background banner
  • Second, add CSS classes in your index.wxml. Your code will look like this: 
<view class="banner">
  <view class="title">Get new superpowers</view>
  <view class="proposition">Learn to code!</view>
  <button type="default">Start today</button>
</view>

Then copy/paste the CSS code below in index.wxss and customize it

.banner {
  color: white;
  text-align: center;
  height: 100vh;
  background-size: cover !important;
  display: flex;
  align-items: center;
  justify-content: center;
  flex-direction: column;
  background-image: url('https://cloud-minapp-13908.cloud.ifanrusercontent.com/1fxsJQzleumTqmCq.jpg');
  background-position: center -30px;
  background-size: cover;
  background-repeat: no-repeat;
  padding:30px;
}
.title {
  margin-top: 100px;
  font-size: 40px;
  padding:0px;
  font-weight: bold;
  text-shadow: 0px 0px 3px rgba(0, 0, 0, 0.6);
}
.proposition {
  font-size: 25px;
  font-weight: bold;
  color: rgb(255, 255, 255);
  text-shadow: 0px 0px 3px rgba(0, 0, 0, 0.6);
  margin: 30px;
}
